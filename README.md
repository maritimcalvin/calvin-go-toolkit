# Calvin's Go Toolkit - Moringa AI Capstone

This repo contains a beginner's "Hello World" HTTP server in Go, as part of my Moringa AI Essentials capstone.

## Installation & Setup
1. Download Go from https://go.dev/dl/ (latest stable, e.g., 1.26).
2. Install for your OS (Windows/macOS/Linux).
3. Verify: `go version`
4. Clone this repo: `git clone https://github.com/maritimcalvin/calvin-go-toolkit.git`
5. Navigate: `cd calvin-go-toolkit`
6. Init module if needed: `go mod init calvin-go-toolkit`

## How to Run
